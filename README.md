# Personal Portfolio Website 🚀✨

A sleek, modern, and fully interactive developer portfolio website designed to showcase professional experience, academic achievements, technical skill stacks, and projects. Developed as part of the **Web Development & Designing Internship** at **Oasis Infobyte** (Task 2).

---

## 📌 Project Objective
The goal of this project was to build a comprehensive, engaging single-page personal portfolio website. It effectively highlights my technical background, projects, certifications, and active community involvements while utilizing creative typography and smooth interactive animations to offer an intuitive user experience.

---

## 🛠️ Tools & Technologies Used
* **HTML5:** Structured semantic markup for clear navigation sections.
* **CSS3:** Custom layout system featuring a grid background design, distinct glow borders, and clean alignment rules.
* **JavaScript (Vanilla):** Implemented for dynamic typewriter text rendering in the hero section and smooth scroll behavior.
* **Google Fonts:** Chosen custom typography to highlight structural emphasis across sections.

---

## 📋 Key Features & Design Layout
The interface is structured around a dark grid theme with dynamic purple and violet accent gradients (`#A855F7`), featuring the following structural sections:

1. **Hero Section:** Dynamically welcomes visitors with a custom typing animation ("Web Developer."), prominent social/contact call-to-actions, and a clean minimalist aesthetic.
2. **About Me:** Highlights academic focus, current skill updates, technical community involvement, and professional motivations.
3. **Academic Journey:** A custom visual timeline block displaying historical markers from secondary education to university.
4. **Tech Stack Grid:** Visually organized pill-tags showcasing competencies across Languages, Tools/Environments, and foundational areas.
5. **Projects Showcase:** Clean cards presenting active systems (e.g., MOM Module, Smart Attendance System, and Arcade Runner Game) paired with tech stack markers and repository reference links.
6. **Certificates & Work Experience:** Showcases professional credentials and ongoing technical internships using a uniform column card layout.
7. **Activities & Contact Block:** Tracks community contributions, alongside a focused "Let's Connect" form block with active social pathways.

---

## 🚀 Steps Performed
1. **Layout Mapping:** Drafted a cohesive navigation architecture prioritizing scannability and structural block partitions.
2. **Dynamic Scripting:** Built a typing script to cycle career focus titles in the main hero view seamlessly.
3. **Card Component Design:** Crafted modular CSS templates for consistent presentation of projects, certificates, and academic metrics.
4. **Typography & Styling Calibration:** Configured contrasting background elements against bright accent tones for readable, visually sharp components.

---

## 🎯 Project Outcome
The resulting application is a highly tailored, production-ready developer portfolio that effectively acts as a comprehensive, visually memorable professional hub.
